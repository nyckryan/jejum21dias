
# Jejum Espiritual 21 Dias

Aplicação em Streamlit para acompanhamento de um jejum espiritual com tema **Noiva de Cristo**.

## Funcionalidades

- Plano de 21 dias
- Versículo do dia
- Contexto histórico
- Aplicação prática
- Checklist de progresso
- Área de oração
- Área de alvos espirituais

## Executar localmente

Instale dependências:

pip install -r requirements.txt

Execute:

streamlit run app.py
